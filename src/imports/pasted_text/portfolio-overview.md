PROJECT OVERVIEW
Design a modern, interactive portfolio website for a digital service provider company offering Web Development, Social Media Management, Graphic Design, and AI Automation/Agents services. The website must be visually stunning, highly animated, and fully functional with excellent user experience.

DESIGN SPECIFICATIONS
1. COLOR PALETTE

Primary: Light, modern colors (Light backgrounds: #F8F9FA, #FFFFFF, #F0F3FF)
Accent Colors: Vibrant but subtle (Teal: #00D4AA, Purple: #7C3AED, Blue: #3B82F6)
Text: Dark grays (#1F2937, #374151) for readability
Overall Tone: Light, clean, and professional with pop-of-color accents

2. TYPOGRAPHY

Headlines: Modern, bold sans-serif (Google Fonts: Poppins, Inter Bold)
Body Text: Clean sans-serif (Google Fonts: Inter, Outfit)
Line Height: Generous spacing for readability


3. PAGE STRUCTURE & COMPONENTS
PAGE 1: HOMEPAGE
A. Hero Section (Above Fold)

Attention-grabbing headline with a witty/clever tagline that makes visitors smile
Include creative, humorous copywriting that speaks to clients in a fun yet professional tone
Subtle animated background elements or gradient
Primary CTA button ("View Our Work" / "Let's Collaborate")
Animated floating shapes or icons representing the 4 services

B. Services Overview Section

4 service cards (Web Dev, Social Media, Graphic Design, AI Automation)
Each card with hover animations (lift up, color change, icon animation)
Brief description for each service
Icons that animate on scroll

C. Heavy Scroll Animation Section - "Our Expertise"

Implement scroll-triggered animations:

Animated counter statistics (Project count, Happy clients, etc.)
Parallax scrolling effects
Images/graphics that fade in, slide in, or scale as user scrolls
Text animations (words appearing letter-by-letter or word-by-word)
Staggered animations for list items



D. Featured Projects/Portfolio Showcase Section

Grid layout (3 columns) showing 6 featured client projects
Each project card includes:

Project thumbnail image
Client name
Service category tag
Brief project description
Hover effect (zoom, overlay with description)


Navigation option/button to "View Full Portfolio" or "Explore All Projects"
Scroll animation: cards slide in from bottom with fade effect

E. Client Testimonials & Social Proof Section

Dedicated testimonial carousel/section featuring:

Client video testimonials (video thumbnails with play buttons)
Client review posts (screenshot of posts/reviews from clients)
Star ratings
Client names and company logos
Smooth carousel animation or grid layout
Auto-play with pause on hover
Animated quote marks and fade-in effects



F. Call-to-Action (CTA) Section

Bold, centered section with primary message
Button to "Start Your Project" or "Get In Touch"
Animated background or decoration elements

G. Footer

Company logo
Quick links to pages
Contact information
Social media icons with hover animations
Newsletter subscription form


PAGE 2: PORTFOLIO / PROJECTS PAGE

Filter options (by service type: Web Dev, Social Media, Design, AI)
Full project grid with detailed cards
Each project includes:

Large featured image
Project title
Category tag
Description
Technologies/tools used
Client logo
View project button


Load more functionality or pagination
Scroll animations for cards appearing


PAGE 3: SERVICES PAGE

Detailed breakdown of each service
Service hero section with banner
Service cards with features
Case studies or examples for each service
Comparison table (optional)
CTA button for each service


PAGE 4: ABOUT US PAGE

Company story/mission statement
Team introduction (if applicable)
Company values/why choose us section
Animated timeline or company milestones
Contact CTA


PAGE 5: CONTACT PAGE

Contact form (Name, Email, Service Interest, Message, Budget Range)
Contact information (Email, Phone, Address if applicable)
Social media links
Success message animation on form submission
Map integration (optional)


4. ANIMATION REQUIREMENTS
Scroll-Triggered Animations:

Elements fade in as they enter viewport
Text slides in from left/right
Images scale up smoothly
Cards stagger in sequence
Counters animate from 0 to final number
Parallax effect for background images
Progress bars fill on scroll

Hover Animations:

Buttons change color and scale slightly
Cards lift up with shadow
Icons rotate or bounce
Links underline with animated effect
Images zoom slightly on hover

Micro-interactions:

Smooth page transitions
Loading animations
Form input focus states
Success/error states for forms
Smooth scrolling behavior
Button click ripple effects


5. SPECIFIC DESIGN ELEMENTS
Unique/Creative Elements:

Custom-designed service icons
Animated gradient backgrounds
Floating animated elements throughout
Creative dividers between sections (waves, diagonal cuts, gradients)
Animated brand logo
Unique navigation menu with smooth transitions

Branding:

Professional yet playful tone (reflected in copy and design)
Consistent spacing and alignment
Clear visual hierarchy
Witty copy that reflects company personality


6. DELIVERABLES

All 5 Pages fully designed with all sections
Interactive Prototype with:

Working navigation between pages
Scroll animations demonstrated
Hover states visible
Interactive elements marked


Component Library for reusable elements
Responsive Design Considerations (note breakpoints for mobile/tablet)
Animation Specifications (timing, easing, trigger points)
Design System Document with colors, typography, spacing rules


7. TECHNICAL NOTES FOR DEVELOPER

Use Scroll-triggered animation libraries (e.g., ScrollTrigger, AOS)
Optimize animations for performance
Ensure accessibility (alt text, keyboard navigation)
Mobile responsiveness priority
Fast loading times despite animations
